# CS 1.2: Assignment 3

This program does the following:
* Counts the number of occurrences of each word in the file
* Prints all of the words and their frequencies

# Example:
```
I write, erase, rewrite
Erase again, and then
A poppy blooms.
```

generates the output:
```
a: 1
again: 1
and: 1
blooms: 1
erase: 2
i: 1
poppy: 1
rewrite: 1
then: 1
write: 1
```
